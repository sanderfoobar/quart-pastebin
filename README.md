quart-pastebin
============

A minimalistic pastebin.com/pastie.org clone based on `quart`.

![https://i.imgur.com/6i5j3BS.png](https://i.imgur.com/6i5j3BS.png)

## Features

- Filesystem based (no database)
- Syntax highlighting
- Paste expiration
- Image uploader
- EXIF removal

## Docker

`docker-compose up` will bind on `127.0.0.1:18200`
